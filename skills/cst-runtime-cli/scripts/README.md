# CST Runtime CLI

CST Studio Suite automation CLI toolchain — 113 atomic commands for modeling, simulation, results, farfield, and optimization. Designed for AI agent orchestration and direct CLI use.

**Requires:** CST Studio Suite 2026 (with Python libraries), Python 3.13+

```powershell
pip install cst-runtime-cli
uv run python -m cst_runtime list-tools
uv run python -m cst_runtime health-check --auto-fix
```
